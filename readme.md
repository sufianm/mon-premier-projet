Les etape de lancement de ton projet le script dans run.sh:

1. Rendre le Script Exécutable

- Utilisez la commande suivante pour donner les droits d’exécution au script :


```bash
chmod +x run.sh
```
2. Exécuter le Script Bash

- Exécutez le script en utilisant la commande :

```bash
./run.sh
```
3. Vérifier le Résultat

- Vous devriez voir un message indiquant le succès ou l’échec de la connexion à l’API GitHub.
